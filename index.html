🔍 Mini-SIEM: PowerShell-Based Log Monitoring

This project provides a lightweight Security Information and Event Management (SIEM) solution built entirely with PowerShell.

📦 Features:

  • ✅ Monitors failed Windows login events (4625)
  • ✅ Alerts on login bursts
  • ✅ Exports HTML reports
  • ✅ Optional pop-up alerts
